1. O que sao operadores em programacao? São símbolos usados para realizar operações com valores e variáveis.

2. Para que servem os operadores aritmeticos?
Cite exemplos.Servem para fazer contas matemáticas no programa.

Exemplos:

+ soma; - subtração; * multiplicação; / divisão;  % resto da divisão.

3. Qual a diferenca entre / e % em JavaScript? O operador / faz a divisão e o operador % mostra o resto da divisão.
Exemplo:
10 / 2 = 5
10 % 3 = 1 (resto da divisão)

4. Como descobrir se um numero e par usando o
operador %? Se o resto da divisão por 2 for igual a 0.
Exemplo: numero % 2 === 0

5. O que e uma expressao em programacao? É uma combinação de valores, variáveis e operadores que produz um resultado.

6. O que significa precedencia de operadores? É a ordem em que os operadores são executados em uma expressão.

7. Por que os parenteses podem mudar o resultado de uma expressao? Porque as operações dentro dos parênteses são executadas primeiro.

8. Qual a diferenca entre =, == e ===? 
= atribui um valor.
== compara apenas o valor.
=== compara valor e tipo.

9. Por que e recomendado usar === em JavaScript? Porque faz uma comparação mais segura, verificando o valor e o tipo dos dados.

10. O que os operadores relacionais retornam: texto, numero ou boolean? Retornam valores booleanos: true ou false.

11. Quando usamos o operador >? E o operador >=? > quando queremos verificar se um valor é maior que outro.
>= quando queremos verificar se é maior ou igual.

12. Quando usamos o operador < ? < quando queremos verificar se um valor é menor que outro.
<= quando queremos verificar se é menor ou igual.

13. Para que serve o operador logico &&? Serve para verificar se duas ou mais condições são verdadeiras ao mesmo tempo.

14. Para que serve o operador logico ||? Serve para verificar se pelo menos uma condição é verdadeira.

15. Para que serve o operador logico !? Serve para inverter um valor lógico.

16. Qual a diferenca entre uma condicao simples e uma condicao composta? 
Simples: possui apenas uma condição.
Composta: possui duas ou mais condições ligadas por operadores lógicos.

17. Explique uma situacao real em que duas condicoes precisam ser verdadeiras ao mesmo tempo. Para entrar em uma prova, o aluno precisa apresentar documento e chegar no horário.

18. Explique uma situacao real em que apenas uma entre duas condicoes precisa ser verdadeira. Uma pessoa pode entrar em um evento apresentando RG ou carteira de motorista.

19. Como o console.log() pode ajudar a encontrar erros no codigo? Ele exibe informações no console, permitindo verificar valores de variáveis e identificar onde está o erro.

20. Por que devemos testar o mesmo programa com valores diferentes? Para garantir que ele funcione corretamente em várias situações e evitar erros inesperados.
