# Pesquisa - Lacos de Repeticao
## 1. O que e uma estrutura de repeticao?
Resposta: É uma estrutura usada para executar um bloco de código várias vezes.
## 2. Para que serve o for?
Resposta: Serve para repetir um código quando já sabemos quantas vezes ele será executado.
## 3. Para que serve o while?
Resposta: Serve para repetir um código enquanto uma condição for verdadeira.
## 4. Para que serve o do-while?
Resposta: Serve para repetir um código pelo menos uma vez e continuar repetindo enquanto a condição for verdadeira.
## 5. Diferenca entre os tres lacos
Resposta: For: usado quando sabemos a quantidade de repetições.
While: usado quando não sabemos quantas vezes vai repetir.
Do-while: executa o código primeiro e depois verifica a condição.
