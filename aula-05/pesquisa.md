# Pesquisa - Arrays e Conceitos

## 1. O que é um array em programação?
Resposta: É uma lista que guarda vários valores em uma única variável.

## 2. Por que é melhor usar um array do que criar várias variáveis separadas, como aluno1, aluno2 e aluno3?
Resposta: Porque facilita organizar e acessar vários dados juntos.

## 3. O que é índice em um array?
Resposta: É a posição de cada item dentro do array.

## 4. Por que o primeiro elemento de um array fica na posição 0?
Resposta: Porque a contagem começa do zero na programação.

## 5. Para que serve o .length?
Resposta: Serve para dizer quantos elementos tem dentro de um array.

## 6. Como acessamos o primeiro elemento de um array chamado alunos?
Resposta: alunos[0].

## 7. Como podemos alterar um valor dentro de um array?
Resposta: Acessando o índice e atribuindo um novo valor.

## 8. Qual a relação entre arrays e laços de repetição?
Resposta: Usamos laços para percorrer todos os itens do array.

## 9. Qual a diferença entre percorrer um array com for e apenas mostrar um item específico?
Resposta: O for mostra todos os itens, o específico mostra só um.

## 10. O que é um acumulador? Dê um exemplo de uso com notas ou valores.
Resposta: É uma variável que vai somando valores, como somar notas de alunos.

## 11. Para que servem operadores aritméticos como +, -, * e / dentro de um programa?
Resposta: Para fazer cálculos matemáticos.

## 12. Para que servem operadores lógicos como && e ||?
Resposta: Para comparar condições.

## 13. Qual a diferença entre if/else e switch?
Resposta: if/else usa condições, switch usa casos fixos.

## 14. O que os métodos push e pop fazem em um array?
Resposta: push adiciona um item e pop remove o último.

## 15. O que é uma matriz e como ela se relaciona com arrays?
Resposta: É um array dentro de outro array (tipo uma tabela).
