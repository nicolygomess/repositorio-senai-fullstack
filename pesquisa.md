# Pesquisa - JavaScript Básico

## 1. O que é programação?
Programação é a forma de dar instruções para o computador realizar tarefas. Através dela, podemos criar sites, aplicativos, jogos e diversas outras tecnologias.

## 2. O que é um algoritmo?
Um algoritmo é uma sequência de passos organizada para resolver um problema ou realizar uma atividade. É como uma receita que seguimos para chegar a um resultado.

## 3. Qual a diferença entre algoritmo e linguagem de programação?
O algoritmo mostra o passo a passo da solução de um problema. A linguagem de programação é usada para escrever esses passos de forma que o computador consiga executá-los.

## 4. O que é JavaScript?
JavaScript é uma linguagem de programação criada para tornar páginas da internet mais interativas, permitindo a criação de efeitos, animações e funcionalidades dinâmicas.

## 5. Onde o JavaScript pode ser utilizado atualmente?
Ele pode ser utilizado em páginas web, sistemas online, aplicativos para celulares, jogos e até mesmo no desenvolvimento de servidores.

## 6. O que é o Node.js?
O Node.js é uma tecnologia que permite rodar JavaScript fora do navegador, diretamente no sistema operacional.

## 7. Qual é a função do Node.js no desenvolvimento com JavaScript?
O Node.js possibilita que os desenvolvedores utilizem JavaScript para criar aplicações completas, incluindo a parte que funciona no servidor.

## 8. O que é o VS Code?
VS Code é um ambiente de desenvolvimento muito popular entre programadores, utilizado para escrever, editar e testar códigos.

## 9. Qual a diferença entre um editor de texto comum e um editor de código?
Enquanto um editor de texto serve apenas para escrever documentos, um editor de código oferece ferramentas que facilitam a programação, como destaque de sintaxe e sugestões automáticas.

## 10. O que é o terminal?
O terminal é uma interface onde podemos digitar comandos para executar programas, acessar arquivos e realizar diversas tarefas no computador.

## 11. Para que serve o comando node arquivo.js?
Esse comando é usado para executar um arquivo JavaScript através do Node.js, mostrando o resultado no terminal.

## 12. O que é uma variável?
Uma variável é um recurso utilizado para guardar informações que poderão ser utilizadas ou alteradas durante a execução do programa.

## 13. Qual a diferença entre let e const?
A principal diferença é que uma variável criada com let pode receber novos valores, enquanto uma criada com const permanece com o mesmo valor durante todo o programa.

## 14. O que são tipos de dados? Cite exemplos.
Tipos de dados são classificações das informações armazenadas em um programa. Alguns exemplos são textos (string), números (number) e valores verdadeiro ou falso (boolean).

## 15. Qual a principal diferença entre JavaScript e TypeScript?
O TypeScript é uma versão do JavaScript que adiciona recursos de tipagem, ajudando a tornar o código mais seguro e fácil de manter.
