# 01 - Pesquisa

## 1. O que é uma estrutura condicional em programação?
É uma estrutura usada para tomar decisões no código. Dependendo da condição, o programa executa uma ação ou outra.

Exemplo:
Se estiver chovendo, levo guarda-chuva. Senão, não levo.

## 2. Qual a diferença entre usar apenas if e usar if/else?
O if executa algo somente se a condição for verdadeira.
O if/else permite escolher entre duas opções: uma se for verdadeira e outra se for falsa.

## 3. Quando usamos else if?
Usamos quando temos várias possibilidades para testar.

Exemplo:
Se nota >= 9 = Excelente
Senão se nota >= 6 = Aprovado
Senão = Reprovado

## 4. O que significa uma condição retornar true ou false?
Significa que a condição foi avaliada.
true = verdadeiro
false = falso

## 5. Qual a diferença entre == e === em JavaScript?
== compara apenas o valor.
=== compara valor e tipo.

O mais recomendado é ===.

## 6. Para que servem &&, || e !?
&& significa E.
Ex: idade >= 18 && temDocumento

|| significa OU.
Ex: temPix || temCartao

! significa NÃO.
Ex: !bloqueado

## 7. O que o operador % faz?
Ele retorna o resto da divisão.

Exemplo:
10 % 2 = 0

É útil para descobrir se um número é par ou ímpar.

## 8. O que é switch case?
É uma estrutura usada para comparar vários casos.

É melhor que vários else if quando há muitas opções fixas.

## 9. Para que servem break e default dentro de um switch?
break encerra o case.
default executa caso nenhum case seja verdadeiro.

## 10. Por que é importante testar o programa com vários valores?
Porque ajuda a verificar se o programa funciona corretamente em diferentes situações.
